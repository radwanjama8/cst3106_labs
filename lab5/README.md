# Yatzy 

## Overview 
This is a overview of the Yatzy game and its rules using HTML and CSS

## Rules 
- Each player can have 3 tries to roll dice (5 dices) for the best possible outcome
- After each roll, you can decide to re-roll or keep any dice
- After each turn, write score in empty box on scorecard OR 0
- Each category can be used only once per game

## Scoring
 -**Upper section (faces 1–6):** Ones, Twos, Threes, Fours, Fives, Sixes  
  - **Bonus:** If the upper subtotal ≥ 63 , add +50.
- **Lower section (combinations):**
  - **One Pair** – sum of the pair  
  - **Two Pairs** – sum of both pairs  
  - **Three of a Kind** – sum of the three dice  
  - **Four of a Kind** – sum of the four dice  
  - **Small Straight (1-2-3-4-5)** – = 15 (sum of all dices)
  - **Large Straight (2-3-4-5-6)** – = 20 (sum of all dices)  
  - **Full House (3 of a kind + a pair)** – sum of all dices 
  - **Chance** – sum of all 5 dices
  - **Yatzy (five of a kind)** – 50

## Game length
- 15 categories and 15 turns per player 

### Final score
- **Final = Upper subtotal + Upper bonus  + Lower subtotal**
