# Yatzy Design System

This design system describes the visual representation of the Yatzy game.

## Colors 

- **Primary Color:** `##0b0b0b`
- **Secondary Color(titles):** `#f00`
- **Text Color:** `#FFF`
- **Border Color:** `#222`
- **Background Color:** `#0b0b0b`
- **Button Color:** `#ff9800`
- **Pip Color:** `#FFF`

## Rationale

- **Black background and white text** : For clear visibility and easy usage of game

- **Red titles** : Draw user attention to main parts

- **white dots on dice** : Easy to over black background

- **Orange action buttons** : Clearly indicate actions 

## Typography

- **Font family:** `system-ui, Arial, sans-serif`
- **Base size:** 16px for body
- **Headings:** h1 24px; h2 20px (red)

## Layout 

- **Max width:** 1000 px
- **Gaps** 10 px
- **Main Grid:** Scores on the left and dice on right side

## Components 

## Section/aside
- **Border:** `1px solid #222222`
- **Radius:** `8px`
- **Padding:** `12px`

## Buttons
- **Padding:** `8px 12px`
- **Radius:** `6px`

## Section Titles
- **Color:** `#FF0000`
- **Margin bottom:** ~8px

## Lists
-**Padding:** 6px 8px
-**Total** color `#FF9800`
